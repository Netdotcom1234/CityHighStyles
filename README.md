## About The Project

[![Product Name Screen Shot][product-screenshot]](https://example.com)

CityHighStyles is a modern and stylish e-commerce front-end for a premium menswear brand. It provides a seamless shopping experience for users to browse and order high-quality clothing. The interface is clean, intuitive, and fully responsive, ensuring a great user experience on any device.

### Built With

* [![Vite][Vite.js]][Vite-url]
* [![JavaScript][JavaScript]][JavaScript-url]
* HTML5
* CSS3

## Getting Started

To get a local copy up and running follow these simple steps.

1. Clone the repo
   ```sh
   git clone https://github.com/your_username/your_project.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the development server
   ```sh
   npm run dev
   ```

<!-- FEATURES -->
## Features

- **Product Filtering:** Easily filter products by category.
- **Quick View:** A modal view for quick product details without leaving the page.
- **WhatsApp Integration:** Seamlessly place orders via WhatsApp.
- **Responsive Design:** Looks great on all devices, from mobile to desktop.
